# AWL Reference

Use this file as a compact reference for STEP 7 AWL/STL edits.

## German And English Mnemonics

- English `A` = German `U`
- English `AN` = German `UN`
- English `OPN` = German `AUF`
- English `JU` = German `SPA`
- English `JC` = German `SPB`
- English `JCN` = German `SPBN`

Prefer `awl-text-sync validate` over guessing when instruction semantics or addressing forms are unclear.

## Address Areas

- German inputs: `E`
- English inputs: `I`
- German outputs: `A`
- English outputs: `Q`
- Merkers: `M`
- Local data: `L`

## Peripheral I/O

- `PEB` / `PIB`
- `PEW` / `PIW`
- `PED` / `PID`
- `PAB` / `PQB`
- `PAW` / `PQW`
- `PAD` / `PQD`

- `VAR_INPUT`
- `VAR_OUTPUT`

- `VAR_TEMP`
- `VAR`

Common composite types:

- `STRUCT`
