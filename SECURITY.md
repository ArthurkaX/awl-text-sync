# Security Policy

## Scope

`awl-text-sync` is a local workflow tool for exported STEP 7 AWL text projects.

This repository does not provide any guarantee that generated output is safe for direct deployment to production PLCs without normal engineering review, import, and compile checks inside STEP 7.

## Reporting A Security Issue

If you find a security-sensitive issue, do not open a public GitHub issue first.

Please contact the maintainer directly and include:

- a short description of the issue
- affected version or commit
- reproduction steps
- example input if needed
- expected impact

## Sensitive Data

Do not publish real customer projects, credentials, plant identifiers, IP addresses, or other sensitive operational data in issues or attachments.

If you need to report a problem, reduce the sample to the smallest safe exported `.AWL` and `.sdf` data needed to reproduce it.

## Support Level

Security reports are handled on a best-effort basis.
